# 🔐 Login Form Project

This is a simple and responsive Login Form project developed using HTML, CSS, and JavaScript. The project is connected to Firebase for authentication and is deployed using GitHub Pages.

## 🚀 Live Demo
👉 https://kancharlapurnaadithya.github.io/loginform/

## 📌 Features
- User-friendly login interface
- Responsive design (works on mobile & desktop)
- Firebase Authentication integration
- Real-time form validation

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Firebase
- Git & GitHub

## 🔥 Firebase Integration
This project uses Firebase for:
- User Authentication
- Secure login handling
